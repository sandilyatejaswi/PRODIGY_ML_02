# Customer Segmentation using K-Means Clustering

## Overview

This project implements K-Means Clustering to segment customers of a retail store based on their annual income and spending score. The objective is to identify distinct customer groups and help businesses understand customer behavior for targeted marketing strategies.

## Dataset

Dataset: Customer Segmentation Dataset

Source:
https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Streamlit (if used)

## Project Workflow

1. Data Loading and Exploration
2. Data Preprocessing
3. Finding Optimal Clusters using Elbow Method
4. Applying K-Means Clustering
5. Visualizing Customer Segments
6. Cluster Analysis

## Features

* Customer data visualization
* Elbow Method for optimal cluster selection
* K-Means clustering implementation
* Interactive customer segmentation visualization
* Cluster-wise customer analysis

## Results

The model successfully segments customers into different groups based on purchasing behavior and spending patterns, helping businesses identify target customer categories.
